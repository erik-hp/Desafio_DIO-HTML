Head

A seção "head" de um documento HTML contém metadados sobre a página. Esses metadados incluem informações como o
título da página, links para arquivos de estilo, scripts, e outras configurações que não são diretamente exibidas ao usuário.

Body

A seção "body" de um documento HTML contém o conteúdo que é visível para os usuários. É aqui que se encontram os elementos
como textos, imagens, links, vídeos, formulários e qualquer outra coisa que compõe a interface visual e interativa da página.

Oque são TAGS?

As tags HTML são elementos usados para marcar e formatar os conteúdos: textual ou visual em documentos HTML.
Cada tag HTML tem um nome e uma sintaxe que determina como ela deve ser aberta e fechada.

Elas são formadas por uma abertura e uma fechadura, delimitadas por um símbolo de menor que (<) e maior que (>), respectivamente.
As tags podem conter atributos, definidos pelo símbolo de igual (=) e separados por espaços. Os atributos podem ter valores atribuídos a eles, que devem estar entre aspas duplas (“).

Tags de Cabeçalho:

As tags de cabeçalho em HTML, são usadas para definir títulos e subtítulos em uma página web. Elas ajudam a organizar o conteúdo de forma hierárquica

  1. <H1></H1>

  2. <H2></H2>

  3. <H3></H3>

  4. <H4></H4>

  5. <H5></H5>
  6. <H6></H6>

Usar as tags de cabeçalho de forma adequada é fundamental para criar uma estrutura clara e bem organizada em suas páginas HTML.

Tags de Texto:

As tags de texto definem textos, estilos de fonte, parágrafos, spans, quebras de linhas, etc.

  • Tag de Parágrafo | <p></p>

  • Tag de Citação | <blockquote></blockquote>

  • Tag de Negrito | <strong></strong>

  • Tag de Itálico | <i></i>

  • Tag de Marcação | <mark></mark>

  • Tag de Redução | <small>

  • Tag de Texto Traçado | <del>

  • Tag de Abreviação | <abbr>

Oque são Atributos?

(voltar)

Os atributos HTML são um modificador de um tipo de elemento HTML.
Com eles, podemos identificar melhor um elemento, informar qual arquivo aquela tag deve utilizar, indicar o tipo de um campo de texto, etc.

Exemplo de alguns:

  • Atributos de Estilo | style="conteudo"

  • Tag de Classe | class="conteudo"

  • Tag de Id | id="conteudo"

  • Tag de Href | href="conteudo"
